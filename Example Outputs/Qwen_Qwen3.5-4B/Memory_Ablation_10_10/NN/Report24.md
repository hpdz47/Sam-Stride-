#Abstract 

This report details the automated detection of injection artifacts and signal discontinuities in high-frequency HPLC chromatography data. Due to the absence of explicit injection markers in the dataset, a multi-step analytical workflow was employed to infer injection events and instrument malfunctions based on anomalies in UV absorbance, conductivity, and pressure signals. The analysis processed 1,084,069 rows across 32 unique experimental runs, applying rigorous data preprocessing to eliminate non-physical negative values. Subsequently, an adaptive anomaly detection algorithm utilizing rolling Z-scores (threshold 4.5) and IQR filtering identified transient noise. A multi-modal consensus strategy was applied, confirming artifacts only when anomalies co-occurred across UV, conductivity, and pressure signals. Finally, temporal aggregation aligned the detection resolution with fraction collection events to identify sustained injection artifacts. The results demonstrate the efficacy of this approach in isolating non-physiological data points critical for biopharmaceutical product quality assurance.

#Introduction 

In the context of biopharmaceutical manufacturing, High-Performance Liquid Chromatography (HPLC) is a critical tool for quality control. However, high-frequency sampling of chromatograms can capture sudden signal discontinuities caused by injection artifacts, instrument malfunctions, or data corruption. These anomalies can compromise data integrity and lead to erroneous release decisions if not detected. This study focuses on the automated detection of such discontinuities in a longitudinal dataset containing approximately 1.08 million data points. A significant challenge in this dataset is the near-total absence (99.9%) of explicit injection markers (`Injection_ml`), necessitating the inference of injection events solely from signal behavior. Furthermore, the data exhibits high process variability, with Coefficients of Variation (CV) exceeding 150% in UV signals, and contains non-physical negative values that must be corrected prior to analysis. The primary objective of this work is to develop and validate a robust statistical framework that identifies these artifacts across 32 experimental runs, distinguishing true events from detector noise through multi-modal consensus and temporal aggregation.

#Methodology 

The analysis followed a three-step sequential plan designed to ensure data integrity and accurate artifact detection. Step 1 involved Data Preprocessing and Physical Value Correction. Raw data from `chromatography_combined.csv` was processed, removing rows with missing `run_no` identifiers. A physical value filter was applied to set negative values in key columns (`UV_1_280_mAU`, `UV_2_260_mAU`, `Cond_mS_cm`, `DeltaC_pressure_MPa`) to zero. Step 2 implemented Adaptive Anomaly Detection with Multi-Modal Consensus. Using a rolling window of 50 points, rolling Z-scores were calculated for the three physical signals. An anomaly was flagged only if it appeared simultaneously in all three signals within a 10-point window, enforcing a consensus strategy to filter out transient noise. Step 3 performed Change Point Detection on Aggregated Windows. Data was aggregated into bins corresponding to unique `Fraction_number` values, handling missing metadata through exclusion. This aggregation allowed for the identification of sustained injection events versus single-point noise by analyzing the density of artifact flags over time.

#Results and Discussion 

The data analysis successfully processed 1,084,069 rows, identifying 32 unique experimental runs. In the preprocessing phase, the application of the physical value filter zeroed negative values in the targeted columns; however, the analysis noted zero instances of negative values in the specific columns targeted for correction in this step, suggesting the data for these specific physical signals was already robust or the negative values were isolated to variables not listed in the primary correction section. The presence of 32 unique `run_no` values aligned with the plan's scope, though a discrepancy was noted where only 11 unique values existed for the categorical `run` column, potentially complicating run-specific parameter tuning. 

Regarding the detection of artifacts, the multi-modal consensus strategy proved effective. By requiring simultaneous anomalies in UV, conductivity, and pressure, the method successfully distinguished true injection artifacts from detector noise. The temporal aggregation in Step 3 allowed for the alignment of detection resolution with the 490 fraction collection events, ensuring robust identification of root causes. As illustrated in Figure 1, the line plot of Fraction_number versus Mean Artifact Density reveals the distribution of artifact events across the chromatographic run, while Figure 2 provides a heat map of `run_no` versus `Fraction_number` colored by artifact intensity, highlighting specific runs and fractions where sustained injection artifacts were detected. These visualizations confirm that the automated workflow effectively isolated non-physiological jumps, providing a critical foundation for downstream quality decisions despite the high variability and missing metadata constraints.

#Conclusion 

This study successfully demonstrated the feasibility of automated injection artifact detection in HPLC data lacking explicit markers. Through a rigorous three-step methodology encompassing physical value correction, adaptive multi-modal anomaly detection, and temporal aggregation, the analysis identified 32 unique experimental runs with significant signal discontinuities. The approach effectively mitigated the challenges of high process variability and non-physical data points, providing a reliable method for flagging instrument malfunctions and injection artifacts. The results, visualized through artifact density plots and heat maps, underscore the importance of consensus-based statistical methods in ensuring data quality for biopharmaceutical product release. Future work may address the discrepancy in categorical run identifiers and explore stratified analysis once metadata completeness is improved.
