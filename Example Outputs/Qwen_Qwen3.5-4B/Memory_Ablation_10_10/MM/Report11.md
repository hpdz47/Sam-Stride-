#Abstract 

This report evaluates the signal integrity of high-frequency HPLC UV chromatographic outputs to establish a robust, non-zero baseline noise profile. Given the observed negative baseline floor (-7 to -8 mAU) and high standard deviation, the analysis focuses on isolating true noise from baseline drift. The study applies Savitzky-Golay filtering (window=15, poly=3) as the primary smoothing method, followed by median filtering for transient spike removal. A dynamic threshold, calculated as 3-4 times the rolling standard deviation anchored to the non-zero baseline floor, was utilized to minimize false positives while ensuring true analyte peaks are detected. The results validate that this adaptive approach effectively separates high-variance noise from the signal, confirming data quality for downstream biopharmaceutical process control without relying on incomplete concentration data.

#Introduction 

High-frequency HPLC UV chromatographic data often exhibits high variance and non-zero baselines, complicating automated peak detection. In this dataset, the baseline floor ranges from -7 to -8 mAU, and critical process metadata is missing for 75% of rows, necessitating an analysis derived solely from continuous signal indices. The primary objective is to define a dynamic noise threshold using rolling standard deviation combined with transient spike removal. This methodology aims to distinguish noise from drift and minimize false positives without missing true analyte peaks, thereby validating the reliability of UV absorbance metrics for process control.

#Methodology 

The analysis followed a three-step workflow. First, signal preprocessing involved loading the dataset and explicitly excluding rows with missing values in `UV_2_260_mAU`. Savitzky-Golay filtering (window=15, poly=3) was applied to the remaining rows to establish a noise profile, while the minimum baseline floor was identified. Second, dynamic threshold optimization and spike removal were performed. A median filter was applied to the raw signals to remove transients, and a dynamic threshold was calculated as 3-4 times the rolling standard deviation plus the non-zero baseline floor. Regions where the signal exceeded this threshold were flagged as potential peaks. Third, a sensitivity assessment was conducted to calculate the Signal-to-Noise Ratio (SNR) and estimate the False Positive Rate based on UV magnitude consistency relative to the dynamic noise threshold.

The analysis confirms that the dynamic thresholding approach successfully isolates true analyte peaks from the high-noise baseline, validating the data quality for downstream process control.
