#Abstract 

This report details a comprehensive Data Integrity and Completeness Audit performed on a high-resolution chromatographic dataset to determine its suitability for downstream machine learning modeling. The primary objective was to identify and quantify data quality failures, specifically focusing on missingness rates, unit consistency, and physical validity of analytical signals. The analysis revealed critical feasibility gaps, including over 65% missingness in key process variables (e.g., `Conc_B_ml`, `pH_ml`) and a 75.68% missingness rate for essential process metadata (`chromatography_stage`, `Sample_Code`). Furthermore, the audit identified severe unit mislabeling artifacts where variables ending in `_ml` contained negative values, which are physically impossible for absorbance measurements. Based on strict biopharmaceutical standards requiring >95% completeness and valid units, the dataset was filtered to a 'Fit for Use' subset containing only five variables and a total of three retained rows. Consequently, the 'Sample-Specific Modeling Linkage Rate' is calculated, highlighting that the current data state is insufficient for robust sample-specific modeling without aggressive imputation, which is explicitly prohibited for high-missingness variables.

#Introduction 

The development of predictive models for biopharmaceutical manufacturing relies heavily on the integrity of the underlying chromatographic data. This study focuses on a dataset comprising HPLC and MS records intended for sample-specific modeling. However, preliminary data exploration indicated significant discrepancies between variable names and their physical units, as well as extensive missing data in critical process metadata. The motivation for this analysis was to rigorously evaluate whether the dataset meets strict biopharmaceutical standards for signal quality, completeness, and unit consistency. The primary goal is to identify specific data integrity failures—such as missingness exceeding 65% in key variables and unit mislabeling—that would render predictive modeling unreliable. By quantifying the feasibility gap for sample-specific modeling and strictly prohibiting aggressive imputation, this analysis aims to provide a definitive decision on the dataset's utility and guide the exclusion of corrupted variables to ensure only valid features are utilized for feature engineering.

#Methodology 

The data analysis workflow consisted of three sequential steps designed to progressively filter the dataset based on defined thresholds. Step 1 executed a Data Integrity and Completeness Audit on the raw `chromatography_combined.csv` file. This involved calculating exact missingness percentages and negative value counts for all variables. Strict thresholds were applied: variables with >65% missingness were flagged for exclusion, and variables with >95% missingness were removed entirely. The analysis also identified unit validity issues, specifically negative values in `_ml` columns which indicated data entry errors. Step 2 focused on Peak Shape and Signal Quality Validation. This step utilized visualizations to distinguish physical noise from data entry errors. Histograms of `UV_1_280_mAU` and `UV_2_260_mAU` were generated with a vertical line at zero to highlight negative outliers. Additionally, a scatter plot of `UV_1_280_mAU` versus `volume_ml` was created to verify if mislabeled variables represented cumulative volume artifacts. Step 3 derived the final 'Fit for Use' dataset. This involved filtering for variables meeting the >95% completeness threshold and valid units (non-negative). Variables failing these criteria, such as `System_flow_ml_min` due to high missingness, were excluded. The final output included a summary of retained rows and the calculation of the 'Sample-Specific Modeling Linkage Rate' to assess the percentage of rows with valid identifiers and signal variables.

#Results and Discussion 

The audit results confirm severe data quality issues that preclude the use of the full dataset for reliable machine learning. The analysis identified that the dataset contains high missingness (>65%) in key analytical variables such as `Conc_B_ml`, `pH_ml`, and `UV_2_260_ml`, alongside a 75.68% missingness rate for process metadata (`chromatography_stage`, `Sample_Code`). This results in a 'Modeling Feasibility Gap' where only a small fraction of rows contain both required metadata fields. Furthermore, the visual analysis presented in Figure 1 provides critical evidence regarding unit mislabeling. The histogram subplots for `UV_1_280_mAU` and `UV_2_260_mAU` clearly display clusters of data points extending below the zero line, which is physically impossible for absorbance measurements. This confirms the presence of negative value artifacts. The scatter plot in Figure 1, correlating `UV_1_280_mAU` with `volume_ml`, demonstrates a strong positive correlation. This supports the hypothesis that the mislabeled `_ml` variables were erroneously populated with absorbance values rather than cumulative volume. Consequently, the 'Fit for Use' dataset was constructed by excluding variables failing the >95% completeness threshold or containing negative values. The final dataset retains only five variables (`UV_1_280_mAU`, `UV_2_260_mAU`, `Cond_mS_cm`, `Conc_B_%`, `pH_pH`) and a total of three retained rows. While the 'Sample-Specific Modeling Linkage Rate' is 100.00% for this sparse subset, this metric is misleading as it does not reflect the inability to model the vast majority of the original data. The primary conclusion is that aggressive imputation is prohibited for high-missingness variables, and the current data state renders predictive modeling unreliable for the bulk of the experimental runs.

#Conclusion 

In conclusion, the rigorous evaluation of the chromatographic dataset reveals that it is currently not fit for use for robust sample-specific machine learning modeling. The dataset suffers from critical integrity failures, including excessive missingness in process metadata and concentration variables, as well as significant unit mislabeling artifacts resulting in physically impossible negative absorbance values. The application of strict biopharmaceutical standards (>95% completeness, non-negative values) has reduced the dataset to a minimal subset of three rows. Therefore, any attempt to proceed with modeling using the full dataset would introduce significant bias and error. Future work must focus on data remediation strategies that do not involve aggressive imputation of high-missingness variables, or the collection of additional experimental runs to ensure sufficient data coverage and unit consistency for reliable predictive modeling.
