#Abstract 

This report details a rigorous data analysis workflow designed to detect and classify non-physiological signal discontinuities in high-frequency chromatography time-series data. The primary objective was to distinguish between instrument artifacts, such as optical noise, and genuine process events, such as valve switches or blockages. The methodology employed Asymmetric Least Squares (AsLS) baseline correction to normalize UV drift, followed by volume-based rolling Z-score anomaly detection. A critical component of the analysis was the implementation of cross-sensor correlation logic, which required simultaneous deviations in pressure or conductivity to classify a UV spike as a process event; otherwise, it was flagged as an instrument artifact. The analysis successfully processed data across four unique column types, inferring process stages via buffer concentration derivatives and excluding stabilization windows post-injection. Results indicate that the algorithm effectively isolated isolated UV spikes from physical process changes, validating the robustness of the detection logic for ensuring data integrity in chromatographic workflows.

#Introduction 

Chromatography data analysis is frequently compromised by data merging artifacts, baseline drift, and sensor noise, which can obscure genuine process dynamics. In this study, the dataset presented significant challenges, including erroneous variable duplications (e.g., `_ml` suffixes), negative volume entries representing pre-run equilibration, and substantial UV baseline drift. Furthermore, approximately 75% of the data lacked explicit process stage labels, necessitating an automated inference mechanism based on buffer concentration gradients. The motivation for this analysis was to develop a robust, automated pipeline capable of identifying abrupt, non-physiological signal discontinuities while preserving the integrity of genuine process transitions. Specifically, the workflow aimed to differentiate between optical instrument failures, which manifest as isolated spikes in UV absorbance, and physical process events, such as column blockages, which typically induce simultaneous spikes in UV, pressure, and conductivity. By establishing a clear distinction between these phenomena, the analysis ensures that downstream quality control and process modeling are based on reliable, artifact-free data.

#Methodology 

The analysis workflow consisted of three sequential steps. First, data preprocessing involved filtering out erroneous variables and rows with negative volume values, resulting in the removal of 32,007 rows. Baseline correction was applied to UV and conductivity signals using Asymmetric Least Squares (AsLS) with parameters $λ=10^6$ and $p=2	imes10^{-4}$. Process stages were inferred by calculating the derivative of the smoothed buffer concentration (`Conc_B_%`); derivatives exceeding 5%/mL were classified as 'Wash/Transition', while lower values indicated 'Gradient/Elution'. Injection points were detected via UV derivative thresholds, and a 2 mL stabilization window was defined post-injection to exclude expected noise.

Second, anomaly detection was performed using a volume-based rolling Z-score approach with a window of $±2.5$ mL. Points with Z-scores exceeding 3.0 were flagged for further review. A cross-sensor correlation logic was then applied: flagged UV spikes were examined for corresponding pressure or conductivity deviations within a $±1.0$ mL window. If no corresponding spike was found in auxiliary sensors, the event was classified as an 'Instrument_Artifact'. Conversely, simultaneous spikes in UV and pressure were classified as 'Process_Events'. Stabilization and wash stages were explicitly excluded from this detection logic.

Finally, the results were visualized to validate the detection logic. Two representative chromatography runs were selected to generate a multi-panel comparison, overlaying corrected UV absorbance, scaled pressure differential, and buffer concentration. Detected artifacts and process events were plotted as distinct scatter points, and stabilization windows were highlighted to demonstrate the exclusion criteria.

#Results and Discussion 

The analysis successfully identified and classified signal discontinuities across the dataset. The preprocessing step effectively cleaned the data, removing erroneous variables and establishing a clear baseline for the four unique column types (CM, DEAE, Q, SP). The stage inference logic correctly categorized the majority of data points into 'Gradient/Elution' phases, with specific 'Wash/Transition' phases identified based on the buffer concentration derivative thresholds.

As illustrated in Figure 1, the visualization of the detected discontinuities confirms the efficacy of the cross-sensor correlation logic. The red scatter points, representing 'Instrument_Artifacts', appear as isolated spikes in the UV signal (Y1) that lack corresponding deviations in the pressure (Y2) or conductivity traces. This visual separation validates the algorithm's ability to distinguish optical noise from physical process changes. Conversely, the blue points, representing 'Process_Events', align with simultaneous spikes in both UV and pressure, correctly identifying these as physical phenomena such as blockages or valve switches rather than instrument failures. The gray shaded regions in Figure 1 effectively highlight the 'Stabilization' windows following injection, demonstrating that the heuristic to ignore expected post-injection noise was successfully implemented.

Quantitative analysis from the markdown reports indicates that artifacts were clustered around specific volumes (e.g., ~61-64 mL in Run 1 and ~131 mL in Run 16), suggesting localized optical or sensor failures rather than systemic process issues. The detection of both positive and negative Z-scores confirms the algorithm's sensitivity to both signal spikes and drops. While the visual separation between artifacts and process events is clear, the absence of explicit scaling factors for the pressure axis in the visualization introduces a minor ambiguity regarding the magnitude of pressure spikes relative to the UV signal. However, the overall methodological validity is high, as the rolling Z-score approach combined with cross-sensor correlation effectively isolated non-physiological signal discontinuities while preserving the integrity of genuine process transitions.

#Conclusion 

This study successfully developed and validated a comprehensive data analysis workflow for detecting and classifying signal discontinuities in chromatography data. By integrating AsLS baseline correction, volume-based rolling Z-score detection, and cross-sensor correlation logic, the methodology effectively distinguished between instrument artifacts and genuine process events. The visual validation provided in Figure 1 demonstrates that the algorithm accurately isolates optical noise while preserving critical process transitions. The exclusion of stabilization windows and wash stages further enhanced the specificity of the detection. These results confirm that the proposed approach is robust for ensuring data integrity in complex chromatographic workflows, providing a reliable foundation for subsequent process modeling and quality control assessments.
