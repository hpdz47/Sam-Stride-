#Abstract 

This study rigorously assessed the integrity of a longitudinal chromatographic dataset comprising 1.08 million rows to ensure its fitness for downstream statistical modeling. Primary objectives included resolving schema errors, correcting negative physical values indicative of baseline artifacts or entry errors, and validating the feasibility of imputing missing concentration data using UV absorbance as a proxy. The analysis identified that the 'Injection_Injection' column was entirely empty and replaced negative volume and concentration values with zero. A critical finding was the high rate of missingness (65.66%) for 'Conc_B_ml', for which a proxy validation strategy was employed. While a strong correlation (>0.8) was observed in specific experimental groups, the overall relationship was heterogeneous and often weak or negative, leading to a conservative decision to mark the majority of missing values as 'Unknown' rather than imputing them. Furthermore, the analysis quantified selection bias, revealing that the distribution of 'Conc_B_%' differed between fully observed and imputed rows. The final feature set prioritized precise volume measurements and normalized absorbance values, resulting in a robust dataset that accurately reflects the 'successful fraction only' nature of the process while mitigating the risk of introducing spurious correlations into downstream models.

#Introduction 

Chromatographic data analysis often encounters significant data quality barriers that can compromise the validity of subsequent statistical modeling and machine learning applications. This dataset, representing longitudinal chromatographic runs, contains approximately 1.08 million rows but is plagued by several integrity issues. These include physical variables exhibiting negative minimum values, which typically arise from baseline subtraction artifacts or manual entry errors, and a high prevalence of missing critical parameters such as 'Conc_B_ml' and 'Injection_ml'. Additionally, schema errors, exemplified by an entirely empty 'Injection_Injection' column, indicate potential data processing failures. The primary motivation for this analysis was to distinguish between data entry artifacts and genuine process failures to prevent selection bias. Specifically, the study aimed to correct negative physical values using defined statistical thresholds, validate the feasibility of imputing missing parameters using fully observed proxies without introducing bias, and construct a validated feature set that accurately reflects the full process lifecycle. The focus area was determined to be the rigorous assessment of data fitness, ensuring that any imputation or correction strategies do not propagate spurious correlations or mask extreme values that occurred during initial error correction phases.

#Methodology 

The data analysis workflow was executed in three distinct phases. First, a Data Integrity and Schema Correction phase was performed on the 'chromatography_combined.csv' file. This involved loading the data to verify the 'Injection_Injection' column (which was found to be entirely empty) and identifying rows with negative physical values for 'volume_ml', 'Conc_B_ml', and 'Conc_B_%'. Negative values were replaced with 0.0 to correct entry errors and baseline artifacts. Second, a Missingness Pattern and Proxy Validation phase was conducted to address the 65.66% missingness in 'Conc_B_ml'. Pearson correlation coefficients were calculated between 'Conc_B_ml' and 'UV_2_260_mAU' across 125 distinct groups defined by 'run_no' and 'chromatography_stage'. A strict threshold of >0.8 was applied; imputation was only performed if the correlation met this threshold within a specific group, otherwise, the value was marked as 'Unknown'. Third, a Selection Bias and Feature Set Construction phase was executed. This involved comparing 'Conc_B_%' distributions between rows with and without 'Conc_B_ml' to quantify bias. Missing 'fraction_volume' values were handled by prioritizing 'fraction_volume_ml_min_precise' over imputed maximums to mitigate selection bias. Finally, outliers were filtered using Z-score thresholds (>3.0) applied to the cleaned dataset, and the final feature set was normalized using the selected volume values.

#Results and Discussion 

The analysis revealed significant data quality challenges that required intervention. Schema errors were successfully resolved, with the empty 'Injection_Injection' column dropped and negative physical values corrected to zero. Regarding missingness, the study confirmed a high rate of missing 'Conc_B_ml' values (65.66%). While a strong correlation (>0.8) was validated within specific experimental groups, allowing for the imputation of 128,589 rows, the majority of groups exhibited weak or negative correlations. Consequently, 711,786 rows were marked as 'Unknown' rather than imputed to prevent the propagation of spurious correlations. A critical discussion point is the persistent selection bias: the distribution of 'Conc_B_%' differs slightly between rows with and without 'Conc_B_ml' (Mean: 15.22 vs. 15.95), suggesting the imputation strategy does not fully recover the true concentration distribution for 'Unknown' cases. Furthermore, the presence of negative values prior to correction indicates that the Z-score outlier threshold (>3.0) was applied to a cleaned dataset, potentially masking extreme values that occurred during the initial error correction phase. The final dataset relies heavily on fully observed variables like 'UV_2_260_mAU' and 'run_no', limiting the ability to model process parameters that remain missing (e.g., 'Injection_ml', 'pH_ml'). This necessitates caution in downstream modeling assumptions regarding process completeness, as the data reflects a 'successful fraction only' nature where missing values likely indicate process failures rather than mere data gaps.

#Conclusion 

In conclusion, this study successfully addressed critical data integrity issues in the chromatographic dataset, transforming a raw file with schema errors and negative artifacts into a validated feature set of 1,084,069 rows. The methodology of correcting negative values and resolving schema errors ensured the physical plausibility of the data. However, the analysis of missingness highlighted the limitations of global imputation strategies; while local imputation using UV absorbance was feasible in specific groups, the heterogeneous correlation across the experimental lifecycle prevented a universal solution. The decision to mark a substantial portion of data as 'Unknown' prioritizes data integrity over completeness, avoiding the introduction of bias. Despite these rigorous corrections, the dataset retains inherent limitations, including a persistent selection bias in concentration distributions and missing process parameters. Future modeling efforts must account for these structural gaps and the 'successful fraction only' nature of the data to ensure accurate representation of the chromatographic process lifecycle.
