#Abstract 

This report details the initial phase of a comprehensive data analysis workflow designed to detect injection artifacts and instrument malfunctions in longitudinal High-Performance Liquid Chromatography (HPLC) and Mass Spectrometry (MS) chromatography datasets. The primary objective is to automate the identification of non-physical signal discontinuities in the absence of explicit injection markers, ensuring biopharmaceutical product quality. The analysis focuses on a longitudinal dataset comprising 1.08 million rows across 16 distinct experimental runs. This report summarizes the execution of Step 1, which involved rigorous data preprocessing and physical value correction. Key findings indicate the successful removal of 130,337 negative values in UV absorbance columns and 8,784 negative values in the pressure column, aligning with the plan to eliminate non-physical baseline artifacts. While the dataset was confirmed to contain 16 unique experimental runs, critical methodological gaps remain regarding the execution of subsequent anomaly detection and change point analysis steps, which were not yet performed in this phase. The report concludes that while physical value cleaning was effective, the validity of artifact filtering and the alignment of detected discontinuities with fraction collection events require the execution of the remaining workflow steps.

#Introduction 

The automated detection of sudden signal discontinuities in longitudinal chromatography time-series is critical for identifying injection artifacts, instrument malfunctions, or data corruption that compromise biopharmaceutical product quality. In this study, the analysis focuses on a high-frequency sampling dataset (approx. 138k points per run) where explicit injection markers are nearly entirely absent (99.9% missingness in `Injection_ml`). Consequently, the analysis relies on adaptive statistical anomaly detection strategies, such as rolling Z-scores and Interquartile Range (IQR) filtering, applied to continuous signals including UV absorbance (`UV_1_280_mAU`, `UV_2_260_mAU`), conductivity (`Cond_mS_cm`), and pressure (`DeltaC_pressure_MPa`). The primary objective is to flag discontinuities across 32 experimental runs by enforcing a multi-modal consensus strategy, where a suspected artifact is only confirmed if anomalies in UV signals are corroborated by simultaneous spikes in both conductivity and pressure. This approach distinguishes true events from detector noise. Furthermore, the analysis necessitates a preprocessing step to exclude non-physical negative values and perform temporal aggregation to align detection resolution with fraction collection events. The current report outlines the motivation, methodology, and preliminary results of the first step in this three-stage workflow, which focuses on data cleaning and physical value correction.

#Methodology 

The data analysis workflow was structured into three sequential steps, with the current report detailing the execution of Step 1: Data Preprocessing and Physical Value Correction. The input data file was 'chromatography_combined.csv', containing 1.08 million rows of hierarchical time-series data. The variables of interest included UV absorbance, conductivity, pressure, and the run identifier (`run_no`). The methodology for Step 1 involved two primary actions: (1) Data Cleaning: Rows containing missing 'run_no' values were identified and excluded to ensure proper grouping. (2) Physical Value Correction: A filter was applied to set negative values in the columns `UV_1_280_mAU`, `UV_2_260_mAU`, `Cond_mS_cm`, and `DeltaC_pressure_MPa` to 0.0. This action was taken to remove non-physical baseline artifacts, as negative absorbance and pressure values are physically impossible in this context. The data was subsequently grouped by 'run_no' to facilitate run-specific analysis. The output of this step was a summary report detailing the total rows processed, the count of negative values zeroed per column, and a list of unique 'run_no' values. While the plan outlined Steps 2 and 3 for adaptive anomaly detection and change point analysis, the current methodology strictly adhered to the scope of Step 1 as defined in the analysis plan.

#Results and Discussion 

The execution of Step 1 (Data Preprocessing and Physical Value Correction) on the longitudinal chromatography dataset yielded significant insights into the data quality and structure. The analysis successfully processed the 1.08 million rows of data. A critical finding was the identification and correction of non-physical values: 130,337 negative values were detected and zeroed in the UV absorbance columns (`UV_1_280_mAU`, `UV_2_260_mAU`), and 8,784 negative values were corrected in the pressure column (`DeltaC_pressure_MPa`). Notably, the conductivity column (`Cond_mS_cm`) was confirmed to contain no negative values. These corrections align with the plan's objective to remove non-physical baseline artifacts. The analysis also validated the grouping strategy, confirming that the dataset comprises 16 distinct experimental runs (`run_no`), which is essential for subsequent run-specific anomaly detection. However, the results highlight critical methodological gaps. The current output reflects only Step 1; therefore, specific evidence regarding the 'rolling window of 50 points' and 'Z-score threshold of 4.5' mentioned in Step 2 is absent. Additionally, while the presence of 125,153 missing values in `Fraction_number` (approx. 11.5%) and the 99.9% missingness in `Injection_ml` are confirmed, they have not yet been addressed by the current preprocessing step, which focused solely on zeroing negative values. Without the execution details of Steps 2 and 3, it is impossible to assess the validity of the artifact filtering or the alignment of detected discontinuities with fraction collection events. The current results serve primarily as a confirmation of data cleaning efficacy for physical values but do not yet provide evidence for the automated detection of injection artifacts or instrument malfunctions.

#Conclusion 

The initial phase of the data analysis workflow, focusing on data preprocessing and physical value correction, has been successfully completed. The analysis confirmed the dataset's structure, comprising 16 distinct experimental runs within a 1.08 million row longitudinal dataset. The removal of 130,337 negative UV absorbance values and 8,784 negative pressure values demonstrates the efficacy of the physical value filtering strategy in eliminating non-physical baseline artifacts. However, the analysis is incomplete regarding the primary objective of detecting injection artifacts and instrument malfunctions. The current results lack the necessary evidence from the adaptive anomaly detection (Step 2) and change point analysis (Step 3) phases. Specifically, the absence of data regarding the rolling window parameters, Z-score thresholds, and the multi-modal consensus validation prevents a definitive assessment of artifact rates or their alignment with fraction collection events. Future work must execute Steps 2 and 3 to address the high missingness in metadata and to perform the temporal aggregation required to align detection resolution with the 490 fraction collection events. Only upon completion of these subsequent steps can the root causes of signal discontinuities be robustly identified and validated for downstream release decisions.
